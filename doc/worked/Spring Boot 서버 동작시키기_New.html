  1. 스프링부트 프로젝트 생성
    • https://start.spring.io/ 에서 아래 순서대로 프로젝트 생성
    • Project : Maven
    • Language : Java
    • Spring Boot : 3.2.4
    • Project Metadata

      Group : dw

      Artifact : firstapp

      Packaging : Jar

      Java : 21

      Lombok

    • 파일 다운로드 완료후 압축해제하고 Github 폴더로 복사

  1. src - main - resources - application.properties 파일 오픈

    application.properties 파일은 스프링앱의 세팅을 기록하는 파일임

    아래 세팅을 복사, 붙여넣기

    포트번호 3307은 본인 데이터베이스 포트번호를 적어야함

    # Data Source
spring.datasource.url=jdbc:mariadb://localhost:3307/testdb
spring.datasource.username=root
spring.datasource.password=root
spring.datasource.driver-class-name=org.mariadb.jdbc.Driver
spring.jpa.hibernate.ddl-auto=update
  1. src - main - java - dw - firstapp - FirstappApplication 파일을 클릭하고 Run 하면 서버 실행됨
